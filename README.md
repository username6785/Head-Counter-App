# Head Counter App

A simple and attractive counter application built using HTML, CSS, and JavaScript. This project allows users to increase, decrease, and reset the count with a clean and responsive user interface.

---

## Features

- Increase Counter
- Decrease Counter
- Reset Counter
- Attractive UI Design
- Responsive Layout
- Beginner Friendly Project

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
